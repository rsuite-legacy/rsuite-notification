# rsuite-notification
`rsuite-notification` 提供了 `Alert` 和 `Nofity` 两个组件。
## 快速开始
### 安装

```javascript
npm install rsuite-notification
```

### 引入less文件：

```
@import "~rsuite-notification/lib/less/index";
```
